# ChatterBot

## 网址

```
https://chatterbot.readthedocs.io/en/stable/quickstart.html
```



## 安装

```
环境：python 3.8
pip install chatterbot
pip install git+git://github.com/gunthercox/ChatterBot.git@master
检查已安装的 ChatterBot 版本
python -m chatterbot --version
```

## 安装报错

```
pip install chatterbot

ERROR: Could not build wheels for preshed, thinc, blis, srsly, which is required to install pyproject.toml-based projects
pip install preshed   可以
pip install thinc     可以
pip install blis
pip install srsly     可以
```
